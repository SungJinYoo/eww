A nifty python debugger.

Eww is an MIT licensed Python library that gives you access to a Python REPL inside your running application.

Adding Eww to your app takes two lines of code: 

import eww
eww.embed()

Then just connect with the provided client: 

basecamp ~: eww
Welcome to the Eww console. Type 'help' at any point for a list of available commands.
Running in PID: 86232 Name: ./demo.py
(eww)

That's it.

Head over to our documentation for more information on getting started.
